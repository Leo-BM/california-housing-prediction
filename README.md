# California Housing — Previsão de Preços 📦 *(arquivado)*

Estudo inicial de regressão com o dataset **California Housing**, iniciado no começo da minha transição para Ciência de Dados. O projeto foi interrompido ainda na fase de estruturação (setup e importações) e está **arquivado como registro de trajetória** — os aprendizados de regressão, pipelines do Scikit-Learn e organização de projeto foram incorporados em trabalhos posteriores e completos:

- **[CancerGuard API](https://github.com/Leo-BM/cancerguard-api)** — classificação com deploy em produção (FastAPI + MLflow + SHAP + Docker + CI/CD)
- **[Document Extraction Pipeline](https://github.com/Leo-BM/document-extraction-pipeline)** — pipeline de extração estruturada de documentos com OCR + ML clássico + LLM
- **[Estudo de CNNs no MNIST](https://github.com/Leo-BM/Estudo-de-Redes-Neurais-Convolucional-utilizando-o-dataset-MNIST)** — experimentos controlados de deep learning
